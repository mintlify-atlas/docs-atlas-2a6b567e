## Plan Tiers Explained

### Free Plan

Perfect for trying out WhatDoc or documenting small personal projects:

- **2 documentation generations** by default
- **5 documentation generations** if you have special access (`has5DocsLimit`)
- Can be extended with **promotional generations** (`promoGenerations`)
- Standard priority queue processing
- Access to basic templates only

**Plan Tier:** `free`

### Pro Plan - Starter ($4.99/mo)

Ideal for individual developers and small teams:

- **Unlimited documentation regenerations**
- **Up to 10 repositories**
- Premium templates included
- Priority queue processing
- Support for private repositories

**Plan Tier:** `499`

### Pro Plan - Growth ($9.99/mo)

Designed for growing teams with larger documentation needs:

- **Unlimited documentation regenerations**
- **Up to 25 repositories**
- Premium templates included
- Priority queue processing
- Support for private repositories

**Plan Tier:** `999`

## Special Features

### Promotional Generations

Free tier users can receive additional documentation generations through promotional campaigns. These are tracked via the `promoGenerations` field and added to your base limit.

### Premium Templates Access

Some users may have `hasPremiumTemplates` enabled, granting access to premium documentation templates even on the free tier. This is typically provided through special partnerships or promotional offers.

## Plan Expiry

Pro subscriptions include an expiry date (`proExpiryDate`). When a Pro plan expires:

- Your account automatically reverts to the Free tier
- Existing documentation remains accessible
- You'll need to upgrade again to generate new documentation beyond the free tier limits

## Need Help Choosing?

Not sure which plan is right for you? Consider:

- **Choose Free** if you're documenting 2-5 personal projects
- **Choose Pro ($4.99)** if you need unlimited regenerations for up to 10 repos
- **Choose Pro ($9.99)** if you're managing documentation for 10+ repositories
